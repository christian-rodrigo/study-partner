# API Testing Plan

## 1. Register user

* POST /api/auth/register

## 2. Login

* POST /api/auth/login

## 3. Get profile

* GET /api/me

## 4. Get swipe candidates

* GET /api/swipe/candidates

## 5. Swipe user

* POST /api/swipe/{targetUserId}

## 6. Get matches

* GET /api/matches

## 7. Send message

* POST /api/chats/{chatId}/messages
