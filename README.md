# Study Partner – Backend

Backend for the Study Partner web application.

This project is developed as a university project using **Java Spring Boot**.

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* PostgreSQL (or H2)
* Maven

## Project Structure

```
study-partner-backend
│
├── backend     # Spring Boot backend application
├── docs        # API documentation and design notes
└── README.md
```

## Running the Backend

```
cd backend
mvn spring-boot:run
```

The server will run on:

```
http://localhost:8080
```
